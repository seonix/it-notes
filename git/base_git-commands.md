Базовые команды git
===================

Репозитории
-----------

Создание нового репозитория:

`git init`

Получение репозитория:

`git clone /путь/к/репозиторию`
`git clone юзер@хост:/путь/к/репозиторию`

Индексация и коммит
-------------------

Индексация изменений:

`git add <file>`
`git add .`

Коммит:

`git commit -m "описание коммита"`
